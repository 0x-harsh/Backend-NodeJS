## MongoDB

- No-SQL Document based Database.
- Strong Support for Aggregation Pipes.
- Works on BSON format.
- Best for Node Applications.

## Architecture of MongoDB

- collection - "users"
- document-1  document-2 inside collection named "users"

## Important Commands of MongoDB

- show dbs
- use <db_name>
- show collections
- db.coll.find()
- db.coll.insert()
