# Nodejs Inroduction

1. Javascript
- a scripting langrage to give interactivity to the website/frontend, the language runs inside the browser, with help of javascript engine.

## JS Engine
- software used to interpret javascript.
Chrome uses V8 (Most Popular)
Firefox uses SpiderMonkey
Safari uses Apple JS Engine

## Nodejs
- Developer Ryan Dahl extraced V8 engine from chrome and embedded it with C++ code, the project was called nodejs.
- now we can run js outside the browser, like in server side
- js can talk to native machine because of C++
- we can create webservers in javascript

*NodeJS is an Runtime environment for JavaScript*
